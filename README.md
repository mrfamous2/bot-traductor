# Bot Traductor para Discord

Bot simple para Discord desarrollado en Python que traduce mensajes automáticamente entre distintos idiomas usando Google Translate.

## Características
- Traducción automática de mensajes
- Soporte para múltiples idiomas
- Configuración simple y ligera
- Desarrollado con Discord.py

## Tecnologías
- Python
- Discord.py
- deep-translator

## Estado
Proyecto personal de aprendizaje enfocado en bots de Discord, APIs y automatización.
